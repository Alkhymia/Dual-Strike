<i>Homepage:</i> <a href="http://pohl-michael.gmxhome.de">http://pohl-michael.gmxhome.de</a></p>
<p>This program allows the configuration of according programmed microcontroller devices over USB.
It generates the user interface based on a configuration definition in a XML format which must apply to the connected device.
It uses a modified version of Objective Development's <a href="http://www.obdev.at/products/vusb/bootloadhid.html">BootloadHID</a> commandline program.</p>
