This is the Dual Strike's version of Objective Development's BootloadHID.

Directories
-----------
-ATmega8:     Bootloader for ATmega8 MCUs.
-ATmega168:   Bootloader for ATmega168 MCUs.
-Commandline: Firmware updater for the bootloader.

See "DualStrike_readme.txt" in these directories for more information.
